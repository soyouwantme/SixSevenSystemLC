###  题目：
基于Android的小型进销存软件设计。

#### 详细功能描述：
小型进销存软件能完成商品的进货，销售，库存管理的基本功能，包括进货计划管理，销售记录与统计，库存盘点与缺货预警等。

#### 主要任务：
利用android平台，设计一个小型进销存软件，有基本的进销存功能。

#### 预期成果或目标：
设计报告和演示软件系统。

#### 编程语言：
Eclipse + Android SDK + ADT
