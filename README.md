# 系统级课程设计 SixSevenSystemLC

#### 题目：基于Android的小型进销存软件设计

#### 详细功能描述：
小型进销存软件能完成商品的进货，销售，库存管理的基本功能，包括进货计划管理，销售记录与统计，库存盘点与缺货预警等。
Based on Android 7.1.1（API 25）
